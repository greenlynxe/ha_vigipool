# 0.3.2

Filtration scheduling usability cleanup.

## Improved

- moved schedule editing entities to configuration instead of the main controls area
- shortened schedule entity names for a cleaner device page
- only expose the next useful empty slot instead of showing every empty slot at once
- hide clear-slot buttons when the corresponding slot is not active

## Result

The device overview stays focused on direct controls, while schedule editing becomes much less noisy and more practical inside Home Assistant.
